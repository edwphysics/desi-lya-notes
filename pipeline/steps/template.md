# Lyman-alpha Pipeline Overview

## Description
High-level overview of the Lyman-alpha analysis pipeline.

## Main Stages

1. Data acquisition and preprocessing  
2. Delta field extraction  
3. Correlation function computation  
4. BAO measurement  

## General Notes

- Each stage may depend on specific DESI tools and environments
- Execution is typically performed in HPC systems (e.g., NERSC)

## Structure

Detailed descriptions of each step can be found in the `steps/` directory.
